
# Calculator

This project was built using Vanilla Javascript and deployed with Turbo 360.

## Demo

https://calculator-yhsrx1.turbo360-vertex.com  
## Deployment

To deploy this project run

```bash
  npm install
```
```bash
  turbo devserver
```

  
## Features

- Basic Calculator utilities
- Dark-Light color mode toggle
- Simple UI
